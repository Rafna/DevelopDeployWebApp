Donate Produce from your Garden!

Welcome to the project Donate Produce. This is a community project to help with fresh food and giving back to the community.

Below is the steps for using this application

    To Donate the produce:
  1. Sign up if you do not have an account
  2. Login if you have an account
  3. Add produce information
  4. Add pick up information
  5. Delete produce information when it is picked up
